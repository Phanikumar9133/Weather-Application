# Weather App

A clean, modern, and responsive weather application built with HTML, CSS, and JavaScript. Displays current weather conditions, hourly/daily forecasts, and allows searching for locations worldwide. Supports light and dark themes with a professional, solid-color design.

## Features

- **Current Weather Display**: Large temperature, location, weather description, and detailed metrics (humidity, wind speed, feels like, pressure, etc.).
- **Dynamic Weather Animations**: Subtle background animations based on current weather conditions.
- **Hourly & Daily Forecast**: Scrollable forecast cards for the upcoming hours/days.
- **City Search with Autocomplete**: Search any city with real-time suggestions powered by OpenWeatherMap API.
- **Favorites & Recent Searches**: Save favorite locations and view recently searched cities in a grid layout.
- **Light & Dark Theme Toggle**: Seamless theme switching with smooth transitions.
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices.
- **Loading Skeletons & Error Handling**: Clean UI feedback during data fetching and graceful error messages.
- **Scrollable Navbar**: Navbar scrolls naturally with the page content for a traditional web app feel.


## Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Media Queries)
- Vanilla JavaScript
- OpenWeatherMap API (Current weather & Forecast)
